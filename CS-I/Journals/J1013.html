Question 1: Why is the formula for determining the number of rows in a truth table related only to the number of inputs, and why is the formula 2^n?

The formula for determining the number of rows in a truth table only relates to the number of inputs because inputs determine the amount of outputs, therefore also determining the number of rows which must exist. The formula is 2^n because for each input there is 2 different possibilities of what it could be (0 or 1), and the n represents the number of inputs which are are present in the truth table. The formula is exponential as more combinations of true/false occur as more inputs are added.
