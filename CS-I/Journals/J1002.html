Question 1: How does the ls command differ from the tree command?

The ls command differs from the tree command in format. The tree command shows everything (all files and directories) in a hierarchical and visual format while the ls command only displays the directories or files under the current directory.

Question 2: How do you choose to use one over the other?

You would use ls when you only want to check what is under your current directory, and you would use tree when you want to check your entire Shell's organization.
