Seven Segment LED Digits

Visual represenatation of the 16 digits which will be displayed by the 7 segment LED.

Four-Bit Binary Input Representation

Assume the 4 inputs are labelled as b3, b2, b1, and b0. The outputs represent the digit which will be displayed on the 7 Segment LED, and each singular segment which must light up to create that digit.

Logic Gates

The canonical representation above represents the four inputs, and each of the seven outputs which must be determined as true or false for the certain digit to be displayed. For example, for the digit 5 to be shown, the input would be 0100, and the output would be 0110011 to make the certain segments which must light up to form a 5 be true. In the case of the 5, the segment a, d, and e will be false and not light up, while the rest of the segments will be true and light up, generating the digit 5 on the chip.
